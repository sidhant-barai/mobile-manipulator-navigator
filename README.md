# Mobile Manipulator Navigator

## 📝 Project Description
This repository documents the development of a mobile robot designed to navigate a specific path and execute a defined task. The project involved:
* **Mechanical Design:** Creating the physical robot chassis and manipulator from scratch.
* **Electronics:** Selecting and integrating sensors, actuators, and microcontrollers.
* **System Logic:** Developing the C++ control algorithms to manage sensor data and motor actuation.

## 🚀 Key Features
* **Autonomous Path Following:** Utilizing IR sensors for precision tracking.
* **Environment Sensing:** Dual ultrasonic sensors for real-time distance and wall-regulation.
* **Power Management:** Comprehensive audit of voltage and current requirements for system stability.
* **Prototyping:** Circuit validation performed via TinkerCAD.

## 📂 Repository Contents
* `/code`: Arduino (.ino) source files.
* `/design`: TinkerCAD schematics and assembly photos.
* `/docs`: Power Management analysis and technical specifications.
* `/video`: Muted demonstration of the autonomous navigation task.
