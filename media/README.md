# Media & Design Documentation

This folder contains the technical design artifacts and performance evidence for the Mobile Manipulator.

## 📐 Design & Kinematics
* **System Design (`Robot Design.png`):** A high-resolution render of the robot's physical architecture, highlighting the laser-cut chassis, dual-motor drivetrain, and sensor placement.
* **Measured Dexterous Workspace (`Dexterous Workspace.png`):** Top-down kinematic analysis showing the **18.5 cm** horizontal reach of the manipulator.
* **Vertical Reach Analysis (`Workspace.png`):** Lateral view demonstrating the **18.5 cm** vertical arc and task-execution strike zone.
