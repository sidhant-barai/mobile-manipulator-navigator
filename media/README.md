# Project Media & Visual Documentation

This folder contains the visual evidence of the robot's design and operational performance.

## 📸 Workspace & Design
The photos in this folder illustrate:
* **Robot Assembly:** The physical layout of the DC motors, sensor placements, and the mounting of the manipulator arm.
* **Workspace Interaction:** Captures of the manipulator arm's reach and the robot's footprint within the testing arena.

## 📹 Performance Demonstration
The included video showcases the final integrated test:
1. **Initial Calibration:** The robot aligns itself using ultrasonic distance regulation.
2. **Autonomous Navigation:** The transition to IR-based line following.
3. **Task Completion:** The arm performing its programmed task upon reaching the target zone.
